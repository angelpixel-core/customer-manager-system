Design System Workspace

This Lookbook workspace documents the shared UI package used by the admin app.

Atomic Levels

  • Atoms: primitive controls (ButtonComponent, InputComponent)
  • Molecules: small interaction groups
  • Organisms: reusable composed sections

Usage Principles

  • Use DesignSystem::UI components for delivery-layer UI.
  • Keep business/domain logic outside components.
  • Prefer semantic variants (:primary, :default) over visual naming.
